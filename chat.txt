Project Summary:

System developed in PHP 8.2 with Symfony 6.4 and MySQL in a dockerized environment

Hexagonal Architecture, DDD (Domain-Driven Design), SOLID principles, and Clean Code practices

Implements Events and Subscriber Pattern

Development guided by TDD (Test-Driven Development) with extensive use of testing

Progressive implementation of CQRS (Command Query Responsibility Segregation) + Event Sourcing is required

Main entities: Certification, CertificationAuthority, CertificationTypeEntity, Company, Discount, Project, ShoppingCart, ShoppingCartItem, User

Objective: Project management system with certifications and commercialization features

Event repository pending completion for Event Sourcing

Focus on code quality with tools such as PHPStan, PHP CS Fixer, Psalm, and PHPCodeSniffer

All responses can be in English, Spanish, or Portuguese, but comments within the code must always, without exception, be in English.
